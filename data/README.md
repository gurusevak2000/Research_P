# Exploring Quantum Machine Learning
**Comparative Study of Classical vs Quantum Algorithms**

## Overview
This MCA research project explores how machine learning can be implemented on quantum systems and how it compares to classical models.

## Tools Used
- Python 3.9+
- Qiskit
- Scikit-learn
- Matplotlib

## Folder Structure
Refer to `/src`, `/notebooks`, and `/report` for organization.

## How to Run
1. Install dependencies:  
   `pip install -r requirements.txt`
2. Run the project:  
   `python main.py`
